# Customer Segmentation using K-Means Clustering

## Objective
Segment customers based on income and spending behavior using Machine Learning.

## Dataset
Mall Customer Dataset

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Machine Learning Technique
- K-Means Clustering
- Elbow Method

## Insights
- Customers were divided into 5 segments
- High-income high-spending customers were identified
- Different customer groups can be targeted differently
